# Інструкція з використання плагіна `kubectl res-stats`

Цей плагін є зручною обгорткою для стандартної команди `kubectl top`. Він збирає статистику використання ресурсів (CPU та оперативної пам'яті) у кластері Kubernetes і виводить її у **CSV-форматі** (значення, розділені комами). Це ідеально підходить для експорту даних у табличні редактори (Excel, Google Sheets) або передачі в інші bash-скрипти для автоматизації.

---
